
# Kuncie Technical Test

Kuncie Technical Test - Create services for cart and checkout business


## Authors

- [@mrdhira](https://github.com/mrdhira)


## Installation

Install project dependecy

```bash
  make dep
  make run-http
```
    
## Running Tests

To run tests, run the following command

```bash
  make test
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/mrdhira/210622-kuncie-tchnl-test
```

Go to the project directory

```bash
  cd 210622-kuncie-tchnl-test
```

Install dependencies

```bash
  make dep
```

Start the server

```bash
  make run-http
```
